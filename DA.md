# Data Analytics Bootcamp Syllabus
## Python | SQL | Statistics | Excel | Power BI
**Duration:** 14 Weeks (3.5 months) | **Weekly commitment:** 8-10 hours  
**Prerequisites:** Basic computer literacy, high school math. No coding experience required.

---

## Course Overview
This integrated program teaches you to extract, clean, analyze, and visualize data using the five most in-demand analytics tools. You will complete 6 projects and one final capstone that mirrors real-world business problems.

## Detailed Weekly Syllabus

### Week 1: Excel Foundations & Data Cleaning
- Cell referencing, named ranges, data types.
- Cleaning: remove duplicates, find/replace, text-to-columns, TRIM, PROPER.
- Conditional formatting for outliers/blanks.
- **Lab:** Clean a messy sales file (500+ rows).

### Week 2: Excel Advanced Analytics
- Logical functions (IF, IFS, AND, OR), lookup functions (XLOOKUP, INDEX-MATCH).
- PivotTables with calculated fields, slicers, timelines.
- Power Query: merge, append, unpivot.
- **Project 1:** Interactive sales dashboard in Excel.

### Week 3: Descriptive Statistics & Visualization Theory
- Mean, median, mode, variance, standard deviation, IQR, percentiles.
- Chart selection guide: bar, line, histogram, scatter, box plot.
- **Lab:** Generate descriptive stats using Excel's Data Analysis ToolPak.

### Week 4: Inferential Statistics & Hypothesis Testing
- Normal distribution, Z-scores, Central Limit Theorem.
- Confidence intervals (mean, proportion).
- Hypothesis testing: p-values, t-tests (one/two-sample), chi-square.
- **Project 2:** A/B testing analysis – t-test on landing page conversion rates.

### Week 5: SQL – Querying & Filtering
- SELECT, FROM, WHERE, ORDER BY, LIMIT.
- Operators: BETWEEN, IN, LIKE, IS NULL.
- **Lab:** Query customer database – filter high-value customers, find missing phone numbers.

### Week 6: SQL – Joins, Aggregations & Subqueries
- Aggregate functions (COUNT, SUM, AVG) with GROUP BY and HAVING.
- INNER, LEFT, RIGHT, FULL OUTER, self-join.
- Subqueries (in WHERE, FROM) and CTEs (WITH clause).
- **Project 3:** E-commerce analysis – monthly revenue, top 5 products, inactive customers.

### Week 7: Integration Week – Excel + SQL
- Import SQL query results into Excel (CSV export/ODBC).
- Build dynamic Excel dashboard with slicers using SQL-aggregated data.
- **Mini-Project:** Sales by region/category – SQL extraction + Excel PivotTable dashboard.

### Week 8: Python Basics & NumPy
- Variables, data types, lists, dictionaries, loops, conditionals.
- Functions, lambda, list comprehensions.
- NumPy: arrays, vectorized operations, random sampling.
- **Lab:** Write functions for mean/median; NumPy array manipulations.

### Week 9: Pandas – Data Wrangling & Cleaning
- Series and DataFrame: reading CSV/Excel, head, info, describe.
- Selecting with .loc, .iloc, boolean filtering.
- Handling missing data: dropna, fillna, interpolate.
- String operations: str.contains, str.split.
- **Lab:** Clean 10,000-row customer dataset – impute missing ages, drop duplicates.

### Week 10: Pandas – Grouping, Merging & Time Series
- Groupby + agg, transform, filter.
- Merging: merge (inner, left, outer), concat, join.
- DateTime parsing, resampling (daily→monthly), rolling windows.
- **Project 4:** Customer churn EDA – merge tables, engineer features, compute churn rates.

### Week 11: Data Visualization in Python & EDA
- Matplotlib: line, bar, scatter, subplots.
- Seaborn: heatmaps, pair plots, catplot, distribution plots.
- Complete EDA workflow: univariate, bivariate, multivariate analysis.
- **Project 5:** Housing dataset EDA – price distributions, correlations, outlier detection.

### Week 12: Power BI – Data Modeling & Power Query
- Connecting to Excel, CSV, SQL Server.
- Power Query (M): merge, append, conditional columns, unpivot.
- Star schema: fact/dimension tables, relationships (1:*, cross-filter direction).
- **Lab:** Import sales DB, create date table, build star schema.

### Week 13: Power BI – DAX & Dashboard Design
- Calculated columns vs. measures.
- Core DAX: SUMX, CALCULATE, FILTER, ALL.
- Time intelligence: TOTALYTD, SAMEPERIODLASTYEAR.
- Interactivity: slicers, bookmarks, drill-through.
- Design principles: layout, color, accessibility.
- **Project 6:** Executive dashboard – YoY growth, drill-down page, key influencers.

### Week 14: Final Capstone Project (All Tools Integrated)
**Scenario:** Logistics company – delivery delays causing customer churn.
- **SQL:** Extract order, driver, traffic data.
- **Python:** Clean data, run t-test (on-time vs. late deliveries on satisfaction), correlation heatmap.
- **Statistics:** 95% confidence intervals for average delay by city zone.
- **Excel:** Cohort retention table (repeat rate before/after delay).
- **Power BI:** Single-page dashboard with map of delay hotspots, KPI cards, root cause bar chart.
- **Deliverable:** 10-12 minute video presentation + all code/files.
